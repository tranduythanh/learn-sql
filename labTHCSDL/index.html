Nội Dung Thực Hành (Thực hành trên SQL Server 2000)

  • Lab1 : Các kiểu dữ liệu cơ bản - Tạo cơ sở dữ liệu - Tạo bảng

    • Các kiểu dữ liệu cơ bản.
    • Tạo cơ sở dữ liệu.
    • Tạo bảng trong cơ sở dữ liệu.
    • Tạo quan hệ giữa các bảng.
    • Bài tập

  • Lab2 : Nhập dữ liệu cho bảng

    • Bài tập

  • Lab3 : Truy vấn đơn giản

    • Nhóm lệnh lựa chọn.
    • Nhóm lệnh thực hiện tính toán.
    • Nhóm lệnh cập nhật dữ liệu.
    • Bài tập .

  • Lab4 : Truy vấn con lồng nhau - Tập hợp

    • Các phép toán trên tập hợp.
    • Quan hệ phần tử tập hợp.
    • So sánh tập hợp.
    • Kiểm tra các quan hệ rỗng.
    • Bài tập.

  • Lab5 : Ngôn ngữ T_SQL ( Transact SQL)- Store Procedure

    • Kiểu dữ liệu.
    • Khai báo biến.
    • Định nghĩa hàm, tham số, kiểu trả về.
    • Cấu trúc điều khiển If ... Else, Case.
    • Vòng lặp While.
    • Cú pháp Store Procedure.
    • Store Procedure có tham số đầu vào.
    • Store Procedure có tham số đầu vào và tham số đầu ra.
    • Bài tập.

  • Lab6 : Bài tập làm thêm

  • Quản lý nhân sự.
  • Đăng kí học phần.
